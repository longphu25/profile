Bộ “kỹ năng” giúp AI agent code như senior dev
Đây là repo cực hot của Matt Pocock, chứa hàng loạt Agent Skills để mở rộng khả năng của Claude Code/Cursor/Copilot như viết PRD, TDD, refactor, triage bug và quản lý Obsidian vault. Repo hiện đã có hơn 18.5k ⭐️ trên GitHub.  ￼
Tính năng nổi bật
* to-prd → biến cuộc trò chuyện thành PRD hoàn chỉnh
* to-issues → chia PRD thành GitHub issues nhỏ
* tdd → workflow Test-Driven Development chuẩn
* triage-issue → điều tra bug + tạo fix plan
* design-an-interface → tạo nhiều phương án thiết kế interface
* obsidian-vault → quản lý note Obsidian bằng AI
* git guardrails → chặn các lệnh Git nguy hiểm trước khi chạy  ￼
Vì sao đáng chú ý?
Thay vì chỉ “chat với AI”, repo này biến AI thành teammate có quy trình làm việc rõ ràng.
→ AI không chỉ trả lời, mà còn biết cách lập kế hoạch, review code và tổ chức dự án như một senior engineer.
