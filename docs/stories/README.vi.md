# Stories

Thư mục này chứa các gói công việc cỡ story, các lát cắt roadmap và các kế
hoạch lịch sử.

## Thư Mục

- `plans/`: các plan về DeepBook Predict, TaskOS, chart trading và modular refactor.

## Cách Dùng

Hãy dùng một story file khi công việc lớn hơn một tiny patch. Một story hữu ích
thường bao gồm:

- mục tiêu
- tài liệu sản phẩm bị ảnh hưởng
- boundary kiến trúc bị ảnh hưởng
- bằng chứng validation
- quyết định còn mở
- trạng thái hoàn thành
