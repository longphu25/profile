# NAVI MCP — Lộ Trình Mở Rộng

Hiện dự án mới dùng 7 trên 37 tool MCP. Phần còn lại mở ra một số hướng mở
rộng rõ ràng:

## Nhóm Tool Chưa Dùng

- **Portfolio Intelligence**: PnL, multi-protocol positions, reward history
- **Bridge**: chain list, token search, quote, status, history
- **Volo Vault Analytics**: APY/TVL history, user positions, user transactions
- **DCA**: order list, order detail, browse all orders
- **Advanced Market Data**: pool detail, market config, price feeds, flash loans
- **Fee Analytics**: borrow fee, protocol fee breakdown
- **Transaction Intelligence**: raw transaction view
- **Token Search**: tra cứu token không chuẩn

## Ưu Tiên

1. Mở rộng dashboard/advisor với portfolio deep-dive
2. Làm plugin riêng cho Volo vault analytics
3. Làm plugin bridge đa chain
4. Làm plugin quản lý DCA
5. Mở rộng market deep-dive và flash-loan/oracle explorer

Nếu hoàn thành đủ các phase, mức bao phủ tool có thể tăng dần từ 19% lên gần
100%.
