# Backlog Harness

Theo dõi các cải tiến có thể giúp agent tiếp theo làm việc nhanh hơn hoặc an
toàn hơn.

| Trạng thái | Hạng mục | Lý do quan trọng |
| --- | --- | --- |
| Open | Thêm bộ kiểm tra liên kết tài liệu | Việc đổi đường dẫn có thể âm thầm làm hỏng các liên kết Obsidian/wiki. |
| Open | Thêm dashboard smoke test | Plugin registration và đường dẫn production rất dễ hồi quy. |
| Open | Thêm metadata trạng thái story | Các plan hiện mô tả hướng đi nhưng chưa hiển thị trạng thái done/blocked/current một cách thống nhất. |
